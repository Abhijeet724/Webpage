This project demonstrates a responsive webpage with a fixed navbar, collapsible left menu, main content area, right panel, and a footer. The page adjusts its scale dynamically based on the screen width.

Features
- Fixed Navbar
- Left Menu (collapsible)
- Main Content Area
- Right-Side Panel
- Footer
- Dynamic Scaling Based on Screen Width

Requirements
- Web Browser (Chrome, Firefox, or any modern browser)
